How to Use Jekyll for GitHub Pages?

May 7, 2025

In this post, I’ll walk you through how I built my own website using GitHub Pages and Jekyll, and share some key steps and lessons I learned along the way.

Setting Up the Project

I created a public GitHub repo called MyWebsite2 and cloned it locally. Then, I created a simple index.html file. After setting GitHub Pages to deploy from the main branch, my website was live!

Installing and Integrating Jekyll

I installed Ruby and Bundler to prepare for Jekyll. Next, I created a new branch:

git checkout --orphan gh-pages

I initialized Jekyll:

jekyll new --skip-bundle .

Then I edited the Gemfile to uncomment and configure the minima theme and plugins.

Build and Local Testing

After running:

bundle exec jekyll build
bundle exec jekyll serve

I verified everything was working locally.

Troubleshooting Common Issues

After pushing my changes, the site appeared broken. I solved this by removing _site from .gitignore and moving its contents to the root directory.

Conclusion

Combining GitHub Pages and Jekyll taught me a lot about static site hosting. I highly recommend this stack for personal blogs and projects!
